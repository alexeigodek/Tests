Prerequisites

  • Access to the demo via your Okta portal — look for the tile named "Boomi MFT - SE Demo" (there are multiple MFT tiles; use this one specifically)
  • A small test file under 1 MB (CSV preferred) for live upload demos in Flow 01 and 05
  • Optional: FileZilla installed for the Flow 05 deep-dive SFTP demo path
  • For Flow 02: There is a separate healthcare upload portal built using Boomi Flow.
    Access it directly at boomi.to/MFTSecureClinicalPortal You can also find the full demo via demos.boomi.com — search the Boomi Demo Library for MFT and open the tile called "Automating Clinical Data Transfers with Boomi MFT". This includes all demo details, assets, visuals, and execution steps.

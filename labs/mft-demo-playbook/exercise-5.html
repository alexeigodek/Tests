Flow 05 — Overview

Manufacturer to Supplier Distribution

A manufacturer uploads into partner-specific paths. MFT routes each file to the correct supplier automatically — with full target isolation.

What This Flow Demonstrates

A manufacturer uploads files into the MFT platform using either the upload widget or SFTP. Files are dropped into supplier-specific paths — BMW, Mazda, or Volkswagen. MFT routes each file to the corresponding supplier's target endpoint. Each supplier gets only their own files and cannot see the others.

Key Features

  • Upload widget with 3 selectable source paths (BMW, Mazda, Volkswagen)
  • FileZilla export for SFTP-native upload path
  • Source-to-target path mapping — each source path routes to a dedicated supplier target
  • Per-supplier target isolation — full multi-tenancy
  • Activity verification with "Ready for Download" state

Architecture

Manufacturer (Upload widget or SFTP push) → Path Selection (BMW / Mazda / Volkswagen) → Flow Routing (source-to-target path mapping) → Suppliers (each gets only their own files)

This is the second flow with a live upload experience. Great for automotive or manufacturing prospects — the BMW / Mazda / VW branding makes the use case immediately recognizable.
Step 1 of 3

Open Flow 05

Review the flow structure and optional FileZilla export path.

  1. 1

    Navigate to Flows → 05 Manufacturer to Supplier. Review the source (manufacturer) and targets (BMW, Mazda, VW). Note how each supplier is a separate org with its own endpoint.

  2. 2

    (Optional — deep dive) On the flows page, click Export to FileZilla. Import the XML into FileZilla to connect and push files via SFTP — good for prospects who prefer a command-line or SFTP-native demo experience.

The FileZilla export is a strong differentiator — one click generates a pre-configured FileZilla site XML. Partners get a zero-configuration SFTP client setup.
Step 2 of 3

Upload Widget & Test File

Use the upload widget to inject a file into a specific supplier path.

Wait for "Completed" before navigating away. Don't dismiss the widget early — larger files may stall. Use a small file (under 1 MB) for reliable results.
  1. 1

    In the Flow 05 sources list, click the Action icon → Upload Files. The widget shows 3 paths: BMW, Mazda, Volkswagen.

  2. 2

    Choose a specific path — e.g., BMW.

  3. 3

    Select a small file and upload. Wait for the "Completed" status before navigating away.

Step 3 of 3

Target Mapping & Activity

Show the path mapping logic, then verify the file landed in the correct supplier folder.

  1. 1

    Navigate to the target side of Flow 05 → Mapping tab. Show how BMW, Mazda, and VW source paths are each mapped to a dedicated supplier target path.

  2. 2

    Navigate to Activity → filter by "Manufacturing to Supplier" → Apply.

  3. 3

    Find your uploaded file in "Ready for Download" state. Confirm it landed in the correct supplier folder (BMW target, not Mazda or VW).

Flow 05 complete. The combination of upload widget path selection + mapping tab + Activity verification is a clean, end-to-end story for supplier distribution use cases.
